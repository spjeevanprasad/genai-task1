## GenAi-Task1

This project is my submission for the Generative AI internship task.

In this task, I used the GPT-2 model to generate text based on a given input. The program takes a prompt from the user and produces a continuation of that text.

I used the transformers library in Python to load the pre-trained model and generate the output.

## Files
- main.py : Python code for text generation
- output.png : Screenshot of the generated result

## How to run
1. Install required libraries:
   pip install transformers torch

2. Run the program:
   python main.py

3. Enter a prompt when asked and view the generated output.
